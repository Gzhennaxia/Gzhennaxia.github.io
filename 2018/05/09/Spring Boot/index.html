Spring Boot

发表于 2018-05-09
  • spring boot的pom文件中有一个parent标签，用与继承父项目的配置

  • spring boot内嵌tomcat，可以以jar包的形式独立运行

  • 入口类XXXApplication上的注解 @SpringBootApplication等价于@SpringBootConfiguration @EnableAutoConfiguration @ComponentScan三个注解的组合

  • spring boot配置文件application.properties或者application.yml,spring推荐使用YAML

    • Yaml中使用缩进表示层级关系缩进只能用空格，空格数目不重要，只要相同层级的元素左侧对齐即可，不要用tab，否者会出现诡异的问题

    • Yaml大小写敏感（区分大小写）

    • 每个key的冒号后面都一定要加一个空格

    • 可以在类上使用@ConfigurationProperties来读取配置文件中的内容

  • 提供@RestController注解，方法上不用再加@ResponseBody注解，默认返回的就是json数据，更激进的支持前后端分离。
