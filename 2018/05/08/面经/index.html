面经

发表于 2018-05-08 | 更新于 2018-06-03 | 分类于 ProJob

北京开科唯识技术有限公司

2018/5/7在猎聘网投的；2018/5/8 下午两点半在深圳南山区高新园的创维大厦面试；

笔试

  1. 给个长整型数返回 x 天 x 时 x 分 x 秒
  2. 计算 2+22+222+2222+22222…
  3. 打印前十行杨辉三角
  4. 二维数组行列转换
  5. 数据库
    1. 什么是事务，什么情况下使用，举例说明一下死锁
    2. 分页查询 SQL 语句
    3. 内连接和外连接的区别
    4. 子查询和连接查询的区别
    5. 数据库优化

面试

  1. 自我介绍
  2. 上家公司主要做什么业务的
  3. 你在公司做什么的
  4. 技术部分，说一下你自己认为比较擅长的
  5. 你是什么时候来深圳的，之前有没有来过？家是哪儿的？
  6. 简历上写的这些框架写的蛮多的，这些到底掌握了多少啊？
  7. 简单说一下nginx反向代理
  8. 学习能力强这个怎么理解
  9. 你数学学的怎么样，好/非常好？
  10. 再问了一次技术方面自己拿手的，说起码要从底层剥离开，全部能讲得通，讲的能让他理解这项技术
  11. 工作经历没问题吧，离职原因
  12. 问起我瞎编的“女朋友”的一些情况
  13. 来深圳的规划
  14. 开科这套笔试题复杂吗？
  15. 笔试题里的排序题你用的什么排序（笔试题里根本没有排序题，怕是根本没看过我的笔试题）
  16. 有没有想过去其他地方，北京啊上海啊这些？
  17. 你觉得深圳好还是西安好（说他自己是西安人）
  18. 如果开科把你录了，你计划待多久？
  19. 出差能接受吗
  20. 这几个项目是在公司做的还是自己做的
  21. 简历的个人技能不需要写那么多，核心的写上去就行了（他给我简历的建议）
  22. 问我薪资要求
  23. 你跟我讲一下Spring Boot的运行原理

教训

  1. 怀疑你学历的时候要强势的反问他，对我的学历有什么问题吗？
  2. 不要紧张。
  3. 不要说自己单独学的技术，要说是边做边学，在工作中用到了，然后学了之后再实践到工作中。
  4. 面试的时候不要有多余的小动作。
  5. 写出来的东西要能熟练的表达出来，不要吞吞吐吐，简单利索。
  6. 简历的个人技能不需要写那么多，核心的写上去就行了（他给我简历的建议）
  7. 综合题考察逻辑的答得太差，不符合一个本科生的水平

深圳四方精创

2018/5/21 14：00 南山区软件园二期11栋B301室 拉钩

面试

  1. springMVC中加一个过滤器的话需要配置哪些东西，实现类需要实现那个接口，有哪些方法？
  2. 项目上线了吗？
  3. 用户量大概有多少？
  4. 项目里哪里用到了AOP，编程式事务没有用过吗？
  5. mybatis用的那个版本的？
  6. mybatis的拦截器用过吗？
  7. string a = “123”; string b = “123”; a==b返回的是true还是false？
  8. 集合有哪些接口和实现类？
  9. hashtable没有用过吗？
  10. web前端写过吗？
  11. MySQL怎么修改表的某个列的列名，语句怎么写，用的什么工具，存储过程写过吗？
  12. 其他语言了解过吗？
  13. 离职原因
  14. 加班接受吗？
  15. 期待薪资

教训

  1. 集合的基本体系不清晰
  2. 简历多打印几份
  3. 项目描述不够熟练
  4. 前端知识要说会用，不要说没用过。
  5. 框架的版本要熟悉。

紫金支点

2018/5/23 早10点 南山区软件产业基地1栋C座20楼 拉勾

笔试

面试

  1. 用的公司自己的语言

教训

  1. 连公司用的语言的名字都没问
  2. 提前多了解了解要面试的公司，看看主要业务等背景。

麦力奇科技

2018/5/24 下午4点 宝安区西乡大道旁正泰大厦1305 拉勾

背景

  1. 主要做H5游戏
  2. 微信小程序

面试

  1. 数学建模是干嘛的？
  2. 怎么理解IoC和AOP的？
  3. 什么是线程安全问题？
  4. 说一下你知道的Linux命令
  5. 你自己搭建过服务器吗？
  6. 讲一下HackNews的这个ranking算法，这个G是个什么参数，有几个指标参与到打分？
  7. 激活邮件是怎么发的，用的那个邮件服务器？
  8. 点赞是怎么同步的，多久同步一次？
  9. 这个消息队列是怎么实现的？
  10. 这个异步是怎么处理的，直接new一个Thread吗？
  11. 事件处理这里用到的是什么设计模式？
  12. 静态代理与动态代理有什么区别？
  13. 动态代理在哪里用到了，使得整个架构显得很优雅？
  14. 你用过那个OKHTTP吗？
  15. 动态代理的好处（）？
  16. 你熟悉的算法跟我讲一个吧，具体算法是怎么样的？
  17. Redis有用到集群，分布式，读写分离吗？
  18. nginx这里很熟吗，它location的匹配规则是什么，它的root的配置？
  19. 后台系统这里熟悉吗？
  20. GitHub有自己的开源的一些东西吗？
  21. 多线程里的读写锁，同步volety
  22. 线程里面的原子性，一致性，可见性知道吗？
  23. HTTP与TCP协议挑一个讲一下。
  24. JVM调优，这个在线上很重要。
  25. maven怎么打不同的包，测试包啊什么的?
  26. 让你做一个高效率的高并发的一级缓存，能做到吗？
  27. 限流怎么做？
  28. 算法这部分也很有用，比如说客服的分配？

总结

  1. 生产者消费者这些是在多线程里面是常用的模式。
  2. JVM调优，这个在线上很重要
